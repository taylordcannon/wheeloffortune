Team 6 Final Project:
Wheel of Fortune/Trivia.
The point of the game is to put the user in the shoes of a contestant on the game show Wheel of Fortune in the last
moments of figuring out what the phrase is.

Install the batch files in the project file and run from command prompt.

The lines 235 and 237 in WoF server are commented out due to database issues. 
	This allows the user to actually play the game, albeit solo.
